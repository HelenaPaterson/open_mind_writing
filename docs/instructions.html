1.1 Setup

1.1.1 Crossrefs

Section links must start with sec- and look like this: Section 1.1.3.

## Section Title {#sec-section-title}

Internal links look like this: @sec-section-title

Figure links must start with fig- and look like this: Figure 1.1.

Table links must start with tbl- and look like this: Table 1.1.

Table 1.1: The authors of this book
first_name last_name
Lisa DeBruine
Daniël Lakens

See the quarto documentation for more information.

1.1.3 Snippets

Snippets in RStudio provide shortcuts to syntax. For example, in an RMarkdown document, type “r” and shift-tab to expand a code chunk.

You can add your own snippets. Under the Tools menu, choose Edit Code Snippets... and paste the following text into the end of the appropriate sections.

1.1.3.1 Markdown

snippet gls
    r glossary("${1:term}")
    
snippet gls2
    r glossary("${1:term}", "${2:display}")
    
snippet h1
    # ${1:title} {#sec-${2:ref}}
    
snippet h2
    ## ${1:title} {#sec-${2:ref}}
    
snippet h3
    ### ${1:title} {#sec-${2:ref}}
    
snippet h4
    #### ${1:title} {#sec-${2:ref}}
    
snippet h5
    ##### ${1:title} {#sec-${2:ref}}

1.2.2 Figures

It is best practice to set a custom ggplot theme, then each subsequent plot will use that theme. You can put this code in R/my_setup.R after loading ggplot2.

Start with a built-in theme and then add any tweaks with the theme() function.

library(ggplot2)

my_theme <- theme_minimal(base_size = 16) + 
            theme(panel.background = element_rect(fill = "red", 
                                                  color = "black", 
                                                  size = 5),
                  panel.grid = element_blank())
Warning: The `size` argument of `element_rect()` is deprecated as of ggplot2 3.4.0.
ℹ Please use the `linewidth` argument instead.
theme_set(my_theme)
ggplot(midwest, aes(popdensity, percollege)) +
  geom_point(alpha = 0.5) +
  labs(x = "Population Density", y = "Percent College Educated")

1.3 Best Practices

1.3.1 Glossary

Books are set up with lightweight glossary functions from the glossary package.

Look up a term from the glossary file with glossary("alpha"): alpha

Display a different value for the term with glossary("alpha", "$\\alpha$"): \(\alpha\)

Use an inline definition instead of the glossary file with glossary("beta", def = "The second letter of the Greek alphabet"): beta

Just show the definition with glossary("p-value", show = "def"): The probability of seeing an effect at least as extreme as what you have, if the real effect was the value you are testing against (e.g., a null effect)

Show the table of terms defined on this page with glossary_table():

term definition
alpha (stats) The cutoff value for making a decision to reject the null hypothesis; (graphics) A value between 0 and 1 used to control the levels of transparency in a plot
beta The second letter of the Greek alphabet
p-value The probability of seeing an effect at least as extreme as what you have, if the real effect was the value you are testing against (e.g., a null effect)
Wickham, H., Bryan, J., & Barrett, M. (2022). Usethis: Automate package and project setup. https://CRAN.R-project.org/package=usethis
